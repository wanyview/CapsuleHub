# CapsuleHub

> AI 时代的知识资产交易所

让讨论产生的知识不再是"一次性结论"，而是可发现、可使用、可演进、可变现的知识资产。

## 🎯 核心定位

**"AI 讨论产生的可验证、可引用、可演进的知识单元"**

- 不是 HuggingFace（模型/数据）
- 不是 arXiv（论文）
- **是：** 胶囊粒度的知识市场 + 价值追踪系统

## ✨ 特性

- 📦 **知识胶囊** - 结构化的知识单元
- 🏆 **DATM 评分** - Truth / Goodness / Beauty / Intelligence
- 🔍 **语义搜索** - 快速发现相关胶囊
- 📊 **价值追踪** - 引用统计、效果验证
- 🔗 **开放集成** - API 服务、外部系统集成

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/wanyview/CapsuleHub
cd CapsuleHub

# 安装依赖
pip install -r requirements.txt

# 初始化演示数据
python scripts/init_demo.py

# 启动服务
python -m uvicorn app.main:app --reload
```

访问 http://localhost:8000/docs 查看 API 文档。

## 📦 核心概念

```
知识胶囊 = 讨论精华产出
├── 核心洞见 (insight)      - 一句话总结
├── 支撑证据 (evidence)     - 3-5条证据链
├── 行动建议 (action_items) - 可执行建议
└── 元数据 (metadata)       - 质量/价值/溯源
```

## 📊 质量评估 (DATM)

| 维度 | 说明 | 权重 |
|------|------|------|
| **Truth** | 真理性 - 事实准确性 | 35% |
| **Goodness** | 良善性 - 价值导向 | 20% |
| **Beauty** | 美感 - 表达优雅 | 15% |
| **Intelligence** | 智能性 - 洞察深度 | 30% |

**综合评分** = DATM均值 × 置信度

**评级标准：**
- A: ≥80 分
- B: 60-79 分
- C: 40-59 分
- D: <40 分
